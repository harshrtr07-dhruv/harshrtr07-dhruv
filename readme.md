Whether diving into low-level microprocessor architecture or spinning up full-stack applications, I love turning logic into impactful solutions.

- 🔭 Currently developing **CampusFix**, a real-time maintenance tracking application using Flutter and Firebase.
- 🌱 Recently built **Ankurit Greens**, a production-level MERN stack e-commerce platform for organic products.
- 👯 Active core member of the **iOS Club** and **K.A.V.A.C.H** (Cybersecurity) at VIT Bhopal.
